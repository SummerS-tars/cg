### 1. 课程事项提醒 (Module: 课程事项提醒)
*   **解决问题**：指导学生进行期末复习，明确考核重点和作业提交要求 [1]。
*   **所属类别**：**期末复习、作业 / Project 相关提醒**。
*   **学习目标**：掌握核心知识点，了解工业级技术文档的学习方法 [1, 2]。
*   **关键概念**：
    *   **考试范围**：题目均源于 PPT，重点考察作业涉及的渲染管线、Blinn-Phong 模型、MVP 矩阵传递、Shader 编写等 [1]。
    *   **推荐资料**：阅读关于《赛博朋克 2077》光线追踪实现的工业级文档，理解其与引擎的关系 [2]。
    *   **考核机制**：作业完成情况是期末成绩的重要组成部分 [2]。
*   **来源**：[1, 2]。

### 2. 参数曲线 (Module: 参数曲线)
*   **解决问题**：解决从字体设计到复杂工业造型的数学表达问题 [2, 3]。
*   **所属类别**：**曲线曲面**。
*   **学习目标**：理解参数曲线的数学定义，掌握基函数混合控制顶点的原理 [3, 4]。
*   **关键概念**：
    *   **应用背景**：TrueType 矢量字体（王选院士的激光照排系统）、动画路径、汽车及建筑形状建模（如大兴机场依赖 CATIA 的曲面建模） [2, 3]。
    *   **数学定义**：使用**基函数 (Basis Function)** 混合**控制顶点 (Control Vertex)** [3]。
    *   **分段参数多项式 (Piecewise Parametric Polynomials)**：即样条曲线 (Spline Curve)，具有易计算、无限次连续、易推导等优点 [4]。
    *   **局部控制 (Local Control)**：修改控制顶点仅影响附近曲线段，不改变整体形状 [4]。
*   **公式**：
    *   一般形式：$x(u) = \sum_{i=0}^{n} B_i(u) \cdot V_{ix}$，$y(u) = \sum_{i=0}^{n} B_i(u) \cdot V_{iy}$ [3]。
    *   线段示例：$B_0(u) = 1-u$，$B_1(u) = u$（一次多项式基函数） [4]。
*   **来源**：[2-4]。

### 3. 渲染管线 (Module: 渲染管线)
*   **解决问题**：将 3D 场景数据（几何、材质、光照）转化为 2D 图像的核心框架问题 [5]。
*   **所属类别**：**渲染管线**。
*   **学习目标**：清晰描述管线流程，理解齐次坐标及高度可编程 API 的应用 [5, 6]。
*   **关键概念**：
    *   **主要阶段**：**应用阶段**（CPU 负责场景管理、数据发送）、**几何处理阶段**（GPU 负责顶点变换、光照计算、图元装配）、**光栅化阶段**（三角形转片元）、**像素处理阶段**（片元着色器计算颜色/采样、深度/模板测试及混合） [5]。
    *   **齐次坐标 (Homogeneous Coordinates)**：引入第四维 $w$ 分量，将平移、旋转、缩放统一为 $4 \times 4$ 矩阵乘法，方便 GPU 加速 [6]。
    *   **可编程 API**：在 Vulkan 或 DX12 中手动创建管线、绑定 Shader、设置描述符集和缓冲区 [6]。
*   **来源**：[5, 6]。

### 4. GPU 驱动的现代渲染技术 (Module: GPU 驱动的渲染)
*   **解决问题**：消除超大规模场景中 CPU-GPU 的通信瓶颈问题 [6, 7]。
*   **所属类别**：**GPU-driven rendering**。
*   **学习目标**：理解将渲染决策从 CPU 转移到 GPU 的核心思想 [6]。
*   **关键概念**：
    *   **Mesh Shader**：直接在 GPU 上处理网格簇 (Meshlet)，自行决定可见性和 LOD，减少 CPU 开销 [7]。
    *   **Visibility Buffer**：通过渲染包含物体 ID 和深度的 Buffer 替代传统 GBuffer，提高遮挡剔除效率 [7]。
*   **例子**：2015 年育碧 (Ubisoft) 工程师在游戏机上实现的 GPU 驱动渲染分享 [7]。
*   **来源**：[6, 7]。

### 5. 光线追踪与路径追踪 (Module: 光线追踪与路径追踪)
*   **解决问题**：实现高质量阴影、镜面反射、折射以及漫反射表面间的间接光照（全局光照）问题 [7, 8]。
*   **所属类别**：**光线追踪 / 路径追踪**。
*   **学习目标**：掌握渲染方程、加速结构以及蒙特卡洛积分在路径追踪中的应用 [8]。
*   **关键概念**：
    *   **渲染方程**：描述场景中光能传递平衡的核心。
    *   **BRDF**：描述物体表面反射特性的四维函数。
    *   **加速结构**：**BVH (层次包围盒)** 和 **KD 树**，将求交复杂度降至 $O(\log n)$ [8]。
    *   **路径追踪 (Path Tracing)**：结合蒙特卡洛方法对多条随机弹射路径进行积分。**俄罗斯轮盘赌**技术用于优化路径递归 [8]。
    *   **光线重建 (Ray Reconstruction)**：利用 AI 模型对路径追踪产生的噪点进行智能降噪（如《赛博朋克 2077》应用），标志着渲染转变为 AI 问题 [8]。
*   **来源**：[7, 8]。
