在计算机图形学期末复习中，**图像合成 (Image Compositing)** 与 **阿尔法混合 (Alpha Blending)** 是核心考点 [1, 2]。

### 核心公式与参数意义

图像合成的基本线性插值公式（也称为 Lerp，Linear Interpolation [3]）定义如下：
$$I = \alpha F + (1 - \alpha) B$$

**参数的视觉与数学意义：**
*   **$I$ (最终合成颜色)**：代表观察者在屏幕上看到的最终像素表现 [1]。
*   **$F$ (前景颜色, Foreground Color)**：代表覆盖在上方物体的颜色 [1]。
*   **$B$ (背景颜色, Background Color)**：代表被前景遮挡的下层物体的颜色 [1]。
*   **$\alpha$ (不透明度, Opacity)**：代表前景物体的遮光能力 [1]。
    *   其取值范围严格修正为 **$\alpha \in [4]$** [1]。
    *   当 $\alpha = 1$ 时，前景完全不透明，视觉上仅显示 $F$ [1]。
    *   当 $\alpha = 0$ 时，前景完全透明，视觉上仅显示 $B$ [1]。
    *   当 $0 < \alpha < 1$ 时，呈现半透明融合效果 [1]。

### 数值计算例题

#### 例题 1：普通前景/背景混合计算
**题目：** 在视频会议的虚拟背景功能中，系统识别出人物前景像素（$F$）的颜色值为 $0.8$（归一化亮度），当前背景图片像素（$B$）的颜色值为 $0.2$。若该像素处于人物边缘，通过 **三分图 (Trimap)** 技术计算得出的不透明度 $\alpha = 0.6$ [5]。请计算合成后的最终像素颜色值 $I$。

**解析：**
根据公式 $I = \alpha F + (1 - \alpha) B$：
1.  代入数值：$I = 0.6 \times 0.8 + (1 - 0.6) \times 0.2$
2.  计算：$I = 0.48 + 0.4 \times 0.2 = 0.48 + 0.08 = 0.56$
**结论：** 最终合成后的像素颜色值为 $0.56$。

#### 例题 2：投影融合重叠区亮带消除
**题目：** 在多投影显示系统中，两台投影机在重叠区域共同投射。若不进行处理，重叠区亮度将是单台投影机的 2 倍（产生亮带）。现利用 **阿尔法混合 (Alpha Blending)** 进行边缘融合（Edge Blending）[1]。
假设左投影机提供的原始背景颜色 $B = 1.0$（最大亮度），右投影机作为叠加的前景 $F = 1.0$。若要使重叠区域中心点的最终亮度 $I$ 保持为 $1.0$，应如何设置右投影机在该点的 $\alpha$ 值？

**解析：**
根据公式 $I = \alpha F + (1 - \alpha) B$：
1.  目标是将重叠区亮度归一化，即设 $I = 1.0$。
2.  代入已知量：$1.0 = \alpha \times 1.0 + (1 - \alpha) \times 1.0$。
3.  在此特殊情境下，若两机贡献相同且需要平滑过渡，通常设置该中心点的 $\alpha = 0.5$。
4.  验证：$I = 0.5 \times 1.0 + 0.5 \times 1.0 = 1.0$。
**结论：** 通过将该点的 $\alpha$ 设置为 $0.5$（即两台投影机各贡献 50% 能量），可消除 2 倍亮度的亮带，实现无缝融合。

### 相关技术补充
*   **预乘阿尔法 (Premultiplied Alpha)**：在存储时预先计算 $F \times \alpha$，这在多层图像叠加和减少合成瑕疵（Artifacts）时具有计算优势 [5]。
*   **抗锯齿 (Antialiasing)**：物体边缘的 $\alpha$ 值通常呈现渐变，以消除离散化带来的锯齿感，使合成效果更逼真 [5]。
