纹理映射（Texture Mapping）是计算机图形学中通过在物体表面“穿衣”来增强细节的核心技术 [1]。以下是基于 Week 9 课堂记录与课件 Lecture 08 的基础讲解：

### 1. 纹理如何增加表面细节
现实世界极其复杂，如果通过增加几何图元（如细小的三角形）来显式模拟所有细节，其计算成本将难以承受 [2]。**纹理映射的思想是利用高效的“视觉欺骗”技巧**，将复杂的视觉信息（通常是 2D 图像）映射到 3D 几何表面上，从而在**不增加额外几何复杂度**的情况下，大幅度丰富表面的颜色变化和视觉特征 [1, 3]。例如，通过一张木纹图片即可让一个平整的矩形面片看起来像是一块精细的木板 [1, 4]。

### 2. 3D 表面到 2D 纹理空间的映射：Parameterization
要将 2D 纹理贴合到 3D 物体上，必须建立从三维表面到二维纹理空间的映射关系，这一过程称为**参数化（Parameterization）** [1, 5]。
*   **映射逻辑**：可以将其想象为将 3D 物体的网格“摊平”或“展开”到平面上的过程 [1, 6]。对于球面或立方体等简单几何体，可以使用球投影或立方体投影；而对于复杂模型，则通常需要将其拆解并拼凑成一张“纹理图集”（Atlas） [6]。
*   **映射步骤**：在扫描转换（Scan Conversion）过程中，渲染管线会将像素坐标（Image Coordinates）映射到建模坐标（Modeling Coordinates），再进一步索引到纹理坐标（Texture Coordinates） [7]。

### 3. UV 坐标及其插值
**UV 坐标**是描述 2D 纹理空间地址的参数。
*   **坐标定义**：每个顶点除了拥有空间位置 $(x, y, z)$ 外，还被分配了一组纹理坐标 $(u, v)$，其取值范围通常在 $[8]$ 之间 [1, 7]。其中 $u$ 代表水平轴，$v$ 代表垂直轴 [9]。
*   **光栅化插值**：在光栅化（Rasterization）阶段，GPU 通过对三角形三个顶点的 UV 坐标进行插值，计算出三角形内部每个片元（Fragment）对应的 UV 坐标 [1, 10]。随后，片元着色器利用这些坐标在纹理图像中查找（Lookup）并获取颜色值 [7, 11]。

### 4. 透视校正插值（Perspective-correct Interpolation）
在纹理映射中，简单的线性插值并不能保证视觉上的正确性。
*   **为什么需要校正**：在屏幕空间（Screen Space）进行直接的线性插值（又称仿射纹理映射）会导致纹理扭曲 [1, 10]。这是因为透视投影是非线性的，物体“近大远小”的特性意味着在屏幕上等距的像素，在 3D 空间中对应的物理距离并不相等 [10]。
*   **实现原理**：为了解决这种扭曲，必须进行**透视校正插值**。该方法利用深度信息（通常是 $1/z$）来修正插值权重，通过在每个像素处进行透视除法（Perspective Divide），确保插值后的纹理坐标在视觉上与 3D 几何特征保持一致 [1, 10]。
